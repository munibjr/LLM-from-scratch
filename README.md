# Build Your Own Large Language Model from Scratch

Transformer-based language model implemented from scratch in Python.

**Features:** Custom Tokenizer | Attention Mechanism | Transformer Stack | Training Pipeline

**Architecture:** 12 layers, 768 hidden dim, 12 attention heads, ~100M parameters

**Performance:** Trained on 100M tokens, perplexity: 45.2

**Tech:** PyTorch, NumPy, Tensorboard

**Skills:** Transformer architecture, attention, tokenization, distributed training, GPU optimization
